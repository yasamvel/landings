— ключевые возможности

Всё для управления
прокси-сервером

Форк X-UI с расширенной поддержкой протоколов, улучшенной стабильностью и учётом трафика по каждому клиенту.

Per-client контроль

Квоты трафика, сроки действия, лимиты IP, онлайн-статус. Шаринг ссылок, QR-коды и подписки в один клик.

Статистика трафика

По каждому inbound, клиенту и outbound. Сброс статистики, графики загрузки — полная картина использования.

Multi-node поддержка

Управляй и масштабируй через единую панель — от одного VPS до роя серверов.

Telegram-бот

Удалённый мониторинг и управление сервером прямо из Telegram. Уведомления, статус, команды.

Маршрутизация

WARP, NordVPN, собственные правила, балансировщики нагрузки и цепочки outbound-прокси.

RESTful API

Полная документация Swagger прямо в панели. Интегрируй с чем угодно.

SQLite / PostgreSQL

Выбор хранилища: SQLite для малых нагрузок, PostgreSQL для крупных инсталляций. Миграция между ними.

Docker + Fail2ban

Готовый docker-compose со встроенным Fail2ban для защиты от перебора IP. Работает из коробки.

— быстрый старт

Установка в одну команду

Готовая команда установки — и у тебя панель с защищённым доступом, случайными логином и портом.

→ Установка последней стабильной версии
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
После установки запусти x-ui — откроется меню управления: запуск/остановка, сброс пароля, SSL-сертификаты и всё остальное.
→ Установка конкретной версии
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh) v3.4.0
Укажи тег версии вместо v3.4.0 — установится именно она.
→ Unattended / Cloud-init установка
XUI_NONINTERACTIVE=1 bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
Без единого вопроса! Логин, пароль и путь генерируются автоматически и сохраняются в /etc/x-ui/install-result.env. Готово для Hetzner, AWS, DO, Vultr и любых облаков.
— docker-развёртывание

Запуск через Docker

Готовый docker-compose с PostgreSQL, Fail2ban и всеми зависимостями.

Быстрый старт (SQLite)

Запуск с SQLite — ничего лишнего, всё из коробки.

docker compose up -d

С PostgreSQL

Для высоких нагрузок и multi-node. Uncomment строки в docker-compose.yml.

docker compose --profile postgres up -d
Важно: NET_ADMIN

Fail2ban использует iptables для блокировки. Убедись, что у контейнера есть cap_add: NET_ADMIN, NET_RAW — иначе баны логируются, но не применяются.
