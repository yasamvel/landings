ArchitectureАрхитектура

Mixture of Experts,
rethought from scratchСмесь экспертов,
переосмысленная с нуля

Hy3 uses a 295B MoE with 21B active parameters per token. Each token activates only 8 of 192 experts through a learned top-8 routing mechanism. This keeps inference fast while maintaining the knowledge capacity of a dense 295B model. Hy3 использует 295B MoE с 21B активных параметров на токен. Каждый токен активирует только 8 из 192 экспертов через обученный механизм маршрутизации top-8. Это обеспечивает быстрый вывод при сохранении емкости знаний плотной модели 295B.

Top-8 RoutingМаршрутизация Top-8

Every input token is routed to the 8 most relevant experts among 192. The router is jointly trained end-to-end with the experts, minimizing load imbalance via auxiliary loss. Каждый входной токен направляется к 8 наиболее подходящим экспертам из 192. Маршрутизатор обучается совместно с экспертами end-to-end, минимизируя дисбаланс нагрузки через вспомогательную потерю.

192 Experts192 эксперта

A large expert pool means each expert can specialize on narrower domains. Sparsity (8/192 active) keeps FLOPs per token comparable to a dense 21B model. Большой пул экспертов позволяет каждому специализироваться в узких областях. Разреженность (8/192 активны) сохраняет FLOPs на токен сопоставимыми с плотной моделью 21B.

262K ContextКонтекст 262K

Extended context window of 262,144 tokens for processing entire codebases, research papers, and long conversations without truncation. RoPE scaling applied. Расширенное окно контекста в 262 144 токена для обработки целых кодовых баз, исследовательских статей и длинных диалогов без усечения. Применено масштабирование RoPE.

Efficient InferenceЭффективный вывод

With only 21B parameters active per forward pass, Hy3 delivers low latency and competitive throughput. Free tier available on OpenRouter for experimentation. С всего 21B активных параметров за проход, Hy3 обеспечивает низкую задержку и конкурентную пропускную способность. Бесплатный уровень доступен на OpenRouter для экспериментов.
